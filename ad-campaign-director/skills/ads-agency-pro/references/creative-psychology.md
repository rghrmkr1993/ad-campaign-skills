# Creative Intelligence — Scoring, Frameworks, Psychology

The creative layer: how to generate, evaluate, and iterate ads. Creative quality drives
50%+ of paid-social performance — this file is the quality bar.

## Creative scoring rubric (score every ad 1-5 on each before upload)

1. **Hook** — does the first 2 seconds / headline stop the target scroller specifically?
2. **Clarity** — is the offer understandable with sound off in 3 seconds?
3. **Emotional trigger** — which one is it deliberately pulling (fear, desire, belonging,
   status, relief, curiosity)? "None" = weak ad.
4. **Pain/benefit balance** — leads with the customer's problem or desired outcome, not
   product features. Features support; benefits sell.
5. **Proof** — testimonial, UGC, numbers, authority, before/after (where compliant).
6. **Offer & urgency** — is there a reason to act NOW that's true (real scarcity/deadline)?
7. **CTA** — single, specific, friction-matched ("Get sample" beats "Learn more" for D2C).
8. **Visual hierarchy** — eye lands on hook → proof → CTA in order; brand visible <3s.
9. **Craft** — typography legible on mobile, color contrast, native-to-feed look; color
   psychology consistent with brand position (trust=blues, energy=warm, premium=restraint).
10. **Distinctiveness** — would the algorithm see this as a NEW concept (different Entity
    ID) or a variation of an existing one?

Score <3 on hook, clarity, or distinctiveness → don't upload; rework.

## Copywriting frameworks (12) — when each wins

| Framework | Skeleton | Best for |
|---|---|---|
| PAS | Problem → Agitate → Solve | Cold audiences, pain-aware categories (default for D2C) |
| AIDA | Attention → Interest → Desire → Action | Long copy, landing pages, VSLs |
| BAB | Before → After → Bridge | Transformation categories (fitness, skincare, education) |
| FAB | Feature → Advantage → Benefit | Tech/spec products, B2B one-pagers |
| 4P | Picture → Promise → Prove → Push | Launch emails, webinar promos |
| QUEST | Qualify → Understand → Educate → Stimulate → Transition | Considered B2B, advertorials |
| StoryBrand | Customer=hero, brand=guide, plan, CTA, stakes | Brand sites, about pages, founder videos |
| Hero's Journey | Ordinary world → struggle → mentor → victory | Founder stories, case-study films |
| ACCA | Awareness → Comprehension → Conviction → Action | Cause/awareness-first products |
| PASTOR | Problem → Amplify → Story → Transform → Offer → Response | Long-form sales pages, email sequences |
| Before-After-Bridge (short) | 2-line version of BAB | Hooks, SMS, WhatsApp |
| Direct offer | Offer → proof → CTA | Retargeting, sales events (skip storytelling for warm audiences) |

Rule: cold traffic gets problem-led frameworks (PAS/BAB); warm traffic gets offer-led.
Never make retargeting audiences re-watch the education act.

## Psychological principles — application + ethical line

| Principle | Application in ads | Line not to cross |
|---|---|---|
| Social proof | Review counts, UGC, "10,000+ customers" | Never fabricate counts/testimonials |
| Scarcity / Loss aversion | Real stock levels, deadline offers, "don't lose X" framing | Fake countdowns/false scarcity = brand-killer + policy risk |
| Authority | Derm-tested, certifications, expert creators | No fake credentials; disclosure rules apply |
| Reciprocity | Free samples, genuinely useful lead magnets, value-first content | — |
| Commitment/consistency | Quiz funnels, micro-conversions before purchase | — |
| Liking/unity | Native creators, community language, diaspora/cultural identity | — |
| Anchoring | Was/now pricing, bundle vs single, premium tier first | Fake MRP inflation is illegal in many geos |
| Framing | "9/10 keep it" vs "1/10 return it"; per-day pricing | — |
| Endowment | Trials, AR try-on, "your routine/your plan" personalization | — |
| Peak-end rule | Strongest proof mid-video + end on offer high note; post-purchase delight | — |
| Availability bias | Vivid specific scenarios over abstract claims | — |
| Halo effect | Production quality and packaging shots transfer to product perception | — |

Doctrine: persuasion techniques amplify a true story. If a technique requires a false
premise, the technique is out — both ethically and because ad-account bans follow.

## Creative generation checklist (per sprint)

Concept matrix = angles × formats. Angles: problem, desire/outcome, social proof, founder,
demo/how-it-works, comparison, objection-crusher, identity/culture, meme/trend (where brand-
safe). Formats: UGC talking-head, b-roll+VO, static+headline, carousel, motion-design.
- 10-15 concepts per campaign; write hook FIRST, then body.
- Every video: 9:16, hook <2s, captions, brand <3s, cut for 6s/15s/30s.
- Localize per language market (native VO > subtitles > translated text, in that order).
- Compliance pass per vertical-playbooks.md before upload.
